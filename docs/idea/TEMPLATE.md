# Idea Template

이 템플릿은 `docs/idea/YYYY_MM_DD_NN_Idea.md` 문서를 만들 때 사용한다.

원칙:
- 아이디어 백로그 메모로 유지한다.
- 아직 확정되지 않은 범위를 적는다.
- 구현 상세, 긴 acceptance criteria, 테스트 절차는 `task`나 `test_guide`로 넘긴다.

```md
# YYYY_MM_DD_NN_Idea

## Note

- 이 문서는 버전 범위 확정 문서가 아니라 아이디어 백로그 메모다.

## Problem

- 지금 무엇이 아쉽거나 막히는지 적는다.

## Idea

- 제안하는 아이디어를 짧고 명확하게 적는다.

## Why It Might Matter

- 사용자 경험이나 제품 방향에서 왜 의미가 있는지 적는다.

## Risks / Open Questions

- 아직 모르는 점, 범위 충돌 가능성, 결정이 필요한 점을 적는다.

## Promotion Rule

- 어떤 조건이면 이 아이디어를 `docs/task` 문서로 승격할지 적는다.

## Links

- 관련 task, figma prompt, PR, 참고 문서가 있으면 연결한다.
```
